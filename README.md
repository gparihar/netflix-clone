# 🎬 Netflix Clone

A modern and responsive Netflix-inspired landing page built using HTML, CSS, and JavaScript. This project recreates the look and feel of Netflix's homepage while demonstrating responsive web design, CSS Grid, Flexbox, and interactive UI elements.

## 🌐 Live Demo

🔗 https://netflix-clone-chi-ten-77.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/gparihar/netflix-clone

---

## ✨ Features

- Responsive Netflix-style UI
- Hero Banner Section
- Navigation Bar
- Trending Movies Section
- Feature Cards Section
- FAQ Section
- Email Subscription CTA
- Footer Section
- Hover Animations
- Mobile-Friendly Design

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/gparihar/netflix-clone.git
```

Navigate into the project folder:

```bash
cd netflix-clone
```

Open `index.html` in your browser.
